## Quickstart guide

* Clone or download this Git repo onto your computer.
* Install [Node.js](https://nodejs.org/en/) if you don't have it yet(Required node latest version and gulp 4).
* install gulp using npm install --save gulp-install
* install gulp sass using npm install gulp sass 

* Go to project directory and run command npm install gulp sass
* Go to project directory and run command npm install gulp 

* Install global gulp cli npm install --global gulp-cli

* Run `npm install`
* Run `gulp` to run the default Gulp task

In this proejct, Gulp is configured to run the following functions:

* Compile the SCSS files to CSS
* Autoprefix and minify the CSS file
* BrowserSync files
* Move final CSS and JS files to the `/dist` folder
